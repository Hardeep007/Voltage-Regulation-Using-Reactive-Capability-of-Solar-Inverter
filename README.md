# Voltage-Regulation-Using-Reactive-Capability-of-Solar-Inverter
Stabilising the Node voltages and maintaining it around 1 p.u. for transmission and distribution grids using Reactive support from Solar Inverter

Files Description:
main.m -> Master file to perform analysis and provide us the results as mentioned in report.
data30.m -> Standart IEEE 30 bus system.
mainnewton.m -> Runs the Load flow analysis using NR method.
lfybus.m -> Form the bus admittance matrix.
busout.m -> Prints the power flow solution on the screen.
lineflow.m -> Computes and displays the line flow and losses.
